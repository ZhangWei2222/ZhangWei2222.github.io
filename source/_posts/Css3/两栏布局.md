左侧宽度固定，右侧自适应

```html
<div class="box" id="box">
    <div class="left">左侧定宽</div>
    <div class="right">右侧自适应</div>
</div>
```



## Flex 布局

```css
#box {
	display: flex;
    display: -webkit-flex;
}
.left {
    width: 100px;
    background: yellow;
}
.right {
    flex: 1;
    background: red;
}
```



## Calc 计算

```css
.left {
    width: 100px;
    background: yellow;
}
.right {
    width: calc(100% - 100px);
    background: red;
}
```



## Float + Margin

```css
.left {
    float: left;
    width: 100px;
    background: yellow;
}
.right {
    margin-left: 100px;
    background: red;
}
```



## Position

```css
#box {
    position: relative;
}
.left {
    position: abosulte;
    left: 0;
    top: 0;
    width: 100px;
    background: yellow;
}
.right {
    position: absolute;
    left: 100px;
    top: 0;
    right: 0;
    width: 100%;
    background: red;
}
```



## Table

```css
#box {
	display: table;
}
#box>div {
    display: table-cell;
}
.left {
    width: 100px;
    background: yellow;
}
.right {
    background: red;
}
```

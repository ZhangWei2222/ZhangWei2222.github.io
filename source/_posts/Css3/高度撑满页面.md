场景描述：需要给某个页面设置背景色，但是背景色出现的高度只有实际内容的高度，需要撑开盒子，高度占满。

```css
html,
body,
#app { // vue 项目需要设置 app 盒子
    height: 100%;
    padding: 0;
    margin: 0;
}
div{ // 给具体的盒子设置高度
    background: #ccc;
    height:100%
}
```

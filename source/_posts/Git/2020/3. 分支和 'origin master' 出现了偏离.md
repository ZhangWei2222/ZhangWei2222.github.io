**提示**

> 位于分支 master
>
> 您的分支和 'origin/master' 出现了偏离，
>
> 并且分别有 1 和 1 处不同的提交。
>
> （使用 "git pull" 来合并远程分支）

假如你想要丢弃你所有的本地改动与提交，可以到服务器上获取最新的版本并将你本地主分支指向到它：

```bash
$ git fetch origin
$ git reset --hard origin/master
```

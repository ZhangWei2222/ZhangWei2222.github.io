> [修饰符列表] class  类名  {
> 	属性； // 描述对象的状态信息
> 	方法； // 描述对象的动作信息
> }



Java 语言中所有的 `class` 都属用 引用数据类型；

每一个类都可以编写主方法，但是一般情况下，一个系统只有一个入口，所以主方法一般写一个，比如有一个  StudentTest.class，里面引用了 Student.class，那个只需要在 StudentTest.class 编写主方法即可

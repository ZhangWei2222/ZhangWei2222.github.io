1. 单行注释

```java
// 单行注释，只注释当前行
```

<!-- more -->

2. 多行注释

```java
/*
多行注释
多行注释
多行注释
多行注释
多行注释
.....
*/
```



3. javadoc注释

```java
/**
*	javadoc注释
*	javadoc注释
*	javadoc注释
*	javadoc注释
*	javadoc注释
*	javadoc注释
*/
注意：这种注释是比较专业的注释，该注释信息会被javadoc.exe工具解析提取并生成帮助文档。
```

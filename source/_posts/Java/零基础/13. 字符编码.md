出现乱码的原因：解码和编码采用的不是同一套对照表

<!-- more -->

**ASCII码：**

'a' --> 97

'A' --> 65

'0' --> 48



**unicode编码：**

统一了全球所有的文字，容量较大，有多种具体实现：UTF-8、UTF-16、UTF-32...

Java 采用的是 unicode编码方式，所以标识符可以用中文。现在实际开发中，一般使用 UTF-8 编码方式较多【统一编码方式】

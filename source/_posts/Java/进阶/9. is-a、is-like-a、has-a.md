is-a：泛化

is-like-a：实现

has-a：关联

<!-- more -->

## is-a

```java
public classAnimal{ 
    public void method1(); 
}
public classDog extendsAnimal { 
    // Dog is a Animal
}
```



## is-like-a

```java
public interface I{ 
    public void method1(); 
} 
public classA implements I{ // A islike a I; 
    public void method1(){ 
        // 实现 
    } 
}
```



## has-a

```java
public classA { // A has a B; 
    private Bb;
}
public classB{
}
```

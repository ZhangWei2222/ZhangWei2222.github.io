## 什么是标识符？

在 Java 源程序中，凡是程序员有权利自己命名的单词都是标识符

标识符可以标识：类名、方法名、变量名、接口名、常量名...

<!-- more -->



## 命名规则

- 以字母、下划线 \_ 或 $ 开头，其后可以是字母、数字、下划线 _ 或 $，不能含有其他字符

- 不能数字开头
- 严格区分大小写
- 不能使用关键字
- 标识符的长度不要超过15个字符



## 命名规范

- 见名知意
- 类名、接口名：首字母大写，后面每个单词首字母大写
- 变量名、方法名、参数名：首字母小写，后面每个单词首字母大写
- 常量名：全部大写

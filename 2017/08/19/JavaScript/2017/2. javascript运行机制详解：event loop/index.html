дёҖгҖҒдёәд»Җд№ҲJavaScriptжҳҜеҚ•зәҝзЁӢпјҹ

JavaScriptиҜӯиЁҖзҡ„дёҖеӨ§зү№зӮ№е°ұжҳҜеҚ•зәҝзЁӢпјҢд№ҹе°ұжҳҜиҜҙпјҢеҗҢдёҖдёӘж—¶й—ҙеҸӘиғҪеҒҡдёҖд»¶дәӢгҖӮеҒҮе®ҡJavaScriptеҗҢж—¶жңүдёӨдёӘзәҝзЁӢпјҢдёҖдёӘзәҝзЁӢеңЁжҹҗдёӘDOMиҠӮзӮ№дёҠж·»еҠ еҶ…е®№пјҢеҸҰдёҖдёӘзәҝзЁӢеҲ йҷӨдәҶиҝҷдёӘиҠӮзӮ№пјҢиҝҷж—¶жөҸи§ҲеҷЁеә”иҜҘд»Ҙе“ӘдёӘзәҝзЁӢдёәеҮҶпјҹ

дәҢгҖҒд»»еҠЎйҳҹеҲ—

еҚ•зәҝзЁӢе°ұж„Ҹе‘ізқҖпјҢжүҖжңүд»»еҠЎйңҖиҰҒжҺ’йҳҹпјҢеүҚдёҖдёӘд»»еҠЎз»“жқҹпјҢжүҚдјҡжү§иЎҢеҗҺдёҖдёӘд»»еҠЎгҖӮеҰӮжһңеүҚдёҖдёӘд»»еҠЎиҖ—ж—¶еҫҲй•ҝпјҢеҗҺдёҖдёӘд»»еҠЎе°ұдёҚеҫ—дёҚдёҖзӣҙзӯүзқҖгҖӮ

JavaScriptиҜӯиЁҖзҡ„и®ҫи®ЎиҖ…ж„ҸиҜҶеҲ°пјҢиҝҷж—¶дё»зәҝзЁӢе®Ңе…ЁеҸҜд»ҘдёҚз®ЎIOи®ҫеӨҮпјҢжҢӮиө·еӨ„дәҺзӯүеҫ…дёӯзҡ„д»»еҠЎпјҢе…ҲиҝҗиЎҢжҺ’еңЁеҗҺйқўзҡ„д»»еҠЎгҖӮзӯүеҲ°IOи®ҫеӨҮиҝ”еӣһдәҶз»“жһңпјҢеҶҚеӣһиҝҮеӨҙпјҢжҠҠжҢӮиө·зҡ„д»»еҠЎз»§з»ӯжү§иЎҢдёӢеҺ»гҖӮ

дәҺжҳҜпјҢжүҖжңүд»»еҠЎеҸҜд»ҘеҲҶжҲҗдёӨз§ҚпјҢдёҖз§ҚжҳҜеҗҢжӯҘд»»еҠЎпјҲsynchronousпјүпјҢеҸҰдёҖз§ҚжҳҜејӮжӯҘд»»еҠЎпјҲasynchronousпјүгҖӮеҗҢжӯҘд»»еҠЎжҢҮзҡ„жҳҜпјҢеңЁдё»зәҝзЁӢдёҠжҺ’йҳҹжү§иЎҢзҡ„д»»еҠЎпјҢеҸӘжңүеүҚдёҖдёӘд»»еҠЎжү§иЎҢе®ҢжҜ•пјҢжүҚиғҪжү§иЎҢеҗҺдёҖдёӘд»»еҠЎпјӣејӮжӯҘд»»еҠЎжҢҮзҡ„жҳҜпјҢдёҚиҝӣе…Ҙдё»зәҝзЁӢгҖҒиҖҢиҝӣе…ҘвҖқд»»еҠЎйҳҹеҲ—вҖқпјҲtask queueпјүзҡ„д»»еҠЎпјҢеҸӘжңүвҖқд»»еҠЎйҳҹеҲ—вҖқйҖҡзҹҘдё»зәҝзЁӢпјҢжҹҗдёӘејӮжӯҘд»»еҠЎеҸҜд»Ҙжү§иЎҢдәҶпјҢиҜҘд»»еҠЎжүҚдјҡиҝӣе…Ҙдё»зәҝзЁӢжү§иЎҢгҖӮ

е…·дҪ“жқҘиҜҙпјҢејӮжӯҘжү§иЎҢзҡ„иҝҗиЎҢжңәеҲ¶еҰӮдёӢгҖӮпјҲеҗҢжӯҘжү§иЎҢд№ҹжҳҜеҰӮжӯӨпјҢеӣ дёәе®ғеҸҜд»Ҙиў«и§ҶдёәжІЎжңүејӮжӯҘд»»еҠЎзҡ„ејӮжӯҘжү§иЎҢгҖӮпјү

пјҲ1пјүжүҖжңүеҗҢжӯҘд»»еҠЎйғҪеңЁдё»зәҝзЁӢдёҠжү§иЎҢпјҢеҪўжҲҗдёҖдёӘжү§иЎҢж ҲпјҲexecution context stackпјүгҖӮ

пјҲ2пјүдё»зәҝзЁӢд№ӢеӨ–пјҢиҝҳеӯҳеңЁдёҖдёӘвҖқд»»еҠЎйҳҹеҲ—вҖқпјҲtask queueпјүгҖӮеҸӘиҰҒејӮжӯҘд»»еҠЎжңүдәҶиҝҗиЎҢз»“жһңпјҢе°ұеңЁвҖқд»»еҠЎйҳҹеҲ—вҖқд№Ӣдёӯж”ҫзҪ®дёҖдёӘдәӢд»¶гҖӮ

пјҲ3пјүдёҖж—ҰвҖқжү§иЎҢж ҲвҖқдёӯзҡ„жүҖжңүеҗҢжӯҘд»»еҠЎжү§иЎҢе®ҢжҜ•пјҢзі»з»ҹе°ұдјҡиҜ»еҸ–вҖқд»»еҠЎйҳҹеҲ—вҖқпјҢзңӢзңӢйҮҢйқўжңүе“ӘдәӣдәӢд»¶гҖӮйӮЈдәӣеҜ№еә”зҡ„ејӮжӯҘд»»еҠЎпјҢдәҺжҳҜз»“жқҹзӯүеҫ…зҠ¶жҖҒпјҢиҝӣе…Ҙжү§иЎҢж ҲпјҢејҖе§Ӣжү§иЎҢгҖӮ

пјҲ4пјүдё»зәҝзЁӢдёҚж–ӯйҮҚеӨҚдёҠйқўзҡ„з¬¬дёүжӯҘгҖӮ

дёүгҖҒдәӢд»¶е’Ңеӣһи°ғеҮҪж•°

вҖңд»»еҠЎйҳҹеҲ—вҖқжҳҜдёҖдёӘдәӢд»¶зҡ„йҳҹеҲ—пјҲд№ҹеҸҜд»ҘзҗҶи§ЈжҲҗж¶ҲжҒҜзҡ„йҳҹеҲ—пјүпјҢIOи®ҫеӨҮе®ҢжҲҗдёҖйЎ№д»»еҠЎпјҢе°ұеңЁвҖқд»»еҠЎйҳҹеҲ—вҖқдёӯж·»еҠ дёҖдёӘдәӢд»¶пјҢиЎЁзӨәзӣёе…ізҡ„ејӮжӯҘд»»еҠЎеҸҜд»Ҙиҝӣе…ҘвҖқжү§иЎҢж ҲвҖқдәҶгҖӮдё»зәҝзЁӢиҜ»еҸ–вҖқд»»еҠЎйҳҹеҲ—вҖқпјҢе°ұжҳҜиҜ»еҸ–йҮҢйқўжңүе“ӘдәӣдәӢд»¶гҖӮ

вҖңд»»еҠЎйҳҹеҲ—вҖқдёӯзҡ„дәӢд»¶пјҢйҷӨдәҶIOи®ҫеӨҮзҡ„дәӢд»¶д»ҘеӨ–пјҢиҝҳеҢ…жӢ¬дёҖдәӣз”ЁжҲ·дә§з”ҹзҡ„дәӢд»¶пјҲжҜ”еҰӮйј ж ҮзӮ№еҮ»гҖҒйЎөйқўж»ҡеҠЁзӯүзӯүпјүгҖӮеҸӘиҰҒжҢҮе®ҡиҝҮеӣһи°ғеҮҪж•°пјҢиҝҷдәӣдәӢд»¶еҸ‘з”ҹж—¶е°ұдјҡиҝӣе…ҘвҖқд»»еҠЎйҳҹеҲ—вҖқпјҢзӯүеҫ…дё»зәҝзЁӢиҜ»еҸ–гҖӮ

жүҖи°“вҖқеӣһи°ғеҮҪж•°вҖқпјҲcallbackпјүпјҢе°ұжҳҜйӮЈдәӣдјҡиў«дё»зәҝзЁӢжҢӮиө·жқҘзҡ„д»Јз ҒгҖӮејӮжӯҘд»»еҠЎеҝ…йЎ»жҢҮе®ҡеӣһи°ғеҮҪж•°пјҢеҪ“дё»зәҝзЁӢејҖе§Ӣжү§иЎҢејӮжӯҘд»»еҠЎпјҢе°ұжҳҜжү§иЎҢеҜ№еә”зҡ„еӣһи°ғеҮҪж•°гҖӮ

вҖңд»»еҠЎйҳҹеҲ—вҖқжҳҜдёҖдёӘе…Ҳиҝӣе…ҲеҮәзҡ„ж•°жҚ®з»“жһ„пјҢжҺ’еңЁеүҚйқўзҡ„дәӢд»¶пјҢдјҳе…Ҳиў«дё»зәҝзЁӢиҜ»еҸ–гҖӮдё»зәҝзЁӢзҡ„иҜ»еҸ–иҝҮзЁӢеҹәжң¬дёҠжҳҜиҮӘеҠЁзҡ„пјҢеҸӘиҰҒжү§иЎҢж ҲдёҖжё…з©әпјҢвҖқд»»еҠЎйҳҹеҲ—вҖқдёҠз¬¬дёҖдҪҚзҡ„дәӢд»¶е°ұиҮӘеҠЁиҝӣе…Ҙдё»зәҝзЁӢгҖӮдҪҶжҳҜпјҢз”ұдәҺеӯҳеңЁеҗҺж–ҮжҸҗеҲ°зҡ„вҖқе®ҡж—¶еҷЁвҖқеҠҹиғҪпјҢдё»зәҝзЁӢйҰ–е…ҲиҰҒжЈҖжҹҘдёҖдёӢжү§иЎҢж—¶й—ҙпјҢжҹҗдәӣдәӢд»¶еҸӘжңүеҲ°дәҶи§„е®ҡзҡ„ж—¶й—ҙпјҢжүҚиғҪиҝ”еӣһдё»зәҝзЁӢгҖӮ

еӣӣгҖҒEvent Loop

дё»зәҝзЁӢд»ҺвҖқд»»еҠЎйҳҹеҲ—вҖқдёӯиҜ»еҸ–дәӢд»¶пјҢиҝҷдёӘиҝҮзЁӢжҳҜеҫӘзҺҜдёҚж–ӯзҡ„пјҢжүҖд»Ҙж•ҙдёӘзҡ„иҝҷз§ҚиҝҗиЎҢжңәеҲ¶еҸҲз§°дёәEvent LoopпјҲдәӢд»¶еҫӘзҺҜпјүгҖӮ

дёҠеӣҫдёӯпјҢдё»зәҝзЁӢиҝҗиЎҢзҡ„ж—¶еҖҷпјҢдә§з”ҹе ҶпјҲheapпјүе’Ңж ҲпјҲstackпјүпјҢж Ҳдёӯзҡ„д»Јз Ғи°ғз”Ёеҗ„з§ҚеӨ–йғЁAPIпјҢе®ғд»¬еңЁвҖқд»»еҠЎйҳҹеҲ—вҖқдёӯеҠ е…Ҙеҗ„з§ҚдәӢд»¶пјҲclickпјҢloadпјҢdoneпјүгҖӮеҸӘиҰҒж Ҳдёӯзҡ„д»Јз Ғжү§иЎҢе®ҢжҜ•пјҢдё»зәҝзЁӢе°ұдјҡеҺ»иҜ»еҸ–вҖқд»»еҠЎйҳҹеҲ—вҖқпјҢдҫқж¬Ўжү§иЎҢйӮЈдәӣдәӢд»¶жүҖеҜ№еә”зҡ„еӣһи°ғеҮҪж•°гҖӮ

жү§иЎҢж Ҳдёӯзҡ„д»Јз ҒпјҲеҗҢжӯҘд»»еҠЎпјүпјҢжҖ»жҳҜеңЁиҜ»еҸ–вҖқд»»еҠЎйҳҹеҲ—вҖқпјҲејӮжӯҘд»»еҠЎпјүд№ӢеүҚжү§иЎҢгҖӮиҜ·зңӢдёӢйқўиҝҷдёӘдҫӢеӯҗгҖӮ

дёҠйқўд»Јз Ғдёӯзҡ„req.sendж–№жі•жҳҜAjaxж“ҚдҪңеҗ‘жңҚеҠЎеҷЁеҸ‘йҖҒж•°жҚ®пјҢе®ғжҳҜдёҖдёӘејӮжӯҘд»»еҠЎпјҢж„Ҹе‘ізқҖеҸӘжңүеҪ“еүҚи„ҡжң¬зҡ„жүҖжңүд»Јз Ғжү§иЎҢе®ҢпјҢзі»з»ҹжүҚдјҡеҺ»иҜ»еҸ–вҖқд»»еҠЎйҳҹеҲ—вҖқгҖӮжүҖд»ҘпјҢе®ғдёҺдёӢйқўзҡ„еҶҷжі•зӯүд»·гҖӮ

д№ҹе°ұжҳҜиҜҙпјҢжҢҮе®ҡеӣһи°ғеҮҪж•°зҡ„йғЁеҲҶпјҲonloadе’ҢonerrorпјүпјҢеңЁsend()ж–№жі•зҡ„еүҚйқўжҲ–еҗҺйқўж— е…ізҙ§иҰҒпјҢеӣ дёәе®ғд»¬еұһдәҺжү§иЎҢж Ҳзҡ„дёҖйғЁеҲҶпјҢзі»з»ҹжҖ»жҳҜжү§иЎҢе®Ңе®ғд»¬пјҢжүҚдјҡеҺ»иҜ»еҸ–вҖқд»»еҠЎйҳҹеҲ—вҖқгҖӮ

дә”гҖҒе®ҡж—¶еҷЁ

йҷӨдәҶж”ҫзҪ®ејӮжӯҘд»»еҠЎзҡ„дәӢд»¶пјҢвҖқд»»еҠЎйҳҹеҲ—вҖқиҝҳеҸҜд»Ҙж”ҫзҪ®е®ҡж—¶дәӢд»¶пјҢеҚіжҢҮе®ҡжҹҗдәӣд»Јз ҒеңЁеӨҡе°‘ж—¶й—ҙд№ӢеҗҺжү§иЎҢгҖӮиҝҷеҸ«еҒҡвҖқе®ҡж—¶еҷЁвҖқпјҲtimerпјүеҠҹиғҪпјҢд№ҹе°ұжҳҜе®ҡж—¶жү§иЎҢзҡ„д»Јз ҒгҖӮ

е®ҡж—¶еҷЁеҠҹиғҪдё»иҰҒз”ұsetTimeout()е’ҢsetInterval()иҝҷдёӨдёӘеҮҪж•°жқҘе®ҢжҲҗпјҢе®ғд»¬зҡ„еҶ…йғЁиҝҗиЎҢжңәеҲ¶е®Ңе…ЁдёҖж ·пјҢеҢәеҲ«еңЁдәҺеүҚиҖ…жҢҮе®ҡзҡ„д»Јз ҒжҳҜдёҖж¬ЎжҖ§жү§иЎҢпјҢеҗҺиҖ…еҲҷдёәеҸҚеӨҚжү§иЎҢгҖӮд»ҘдёӢдё»иҰҒи®Ёи®әsetTimeout()гҖӮ

setTimeout()жҺҘеҸ—дёӨдёӘеҸӮж•°пјҢз¬¬дёҖдёӘжҳҜеӣһи°ғеҮҪж•°пјҢз¬¬дәҢдёӘжҳҜжҺЁиҝҹжү§иЎҢзҡ„жҜ«з§’ж•°гҖӮ

дёҠйқўд»Јз Ғзҡ„жү§иЎҢз»“жһңжҳҜ1пјҢ3пјҢ2пјҢеӣ дёәsetTimeout()е°Ҷз¬¬дәҢиЎҢжҺЁиҝҹеҲ°1000жҜ«з§’д№ӢеҗҺжү§иЎҢгҖӮ

еҰӮжһңе°ҶsetTimeout()зҡ„з¬¬дәҢдёӘеҸӮж•°и®ҫдёә0пјҢе°ұиЎЁзӨәеҪ“еүҚд»Јз Ғжү§иЎҢе®ҢпјҲжү§иЎҢж Ҳжё…з©әпјүд»ҘеҗҺпјҢз«ӢеҚіжү§иЎҢпјҲ0жҜ«з§’й—ҙйҡ”пјүжҢҮе®ҡзҡ„еӣһи°ғеҮҪж•°гҖӮ

дёҠйқўд»Јз Ғзҡ„жү§иЎҢз»“жһңжҖ»жҳҜ2пјҢ1пјҢеӣ дёәеҸӘжңүеңЁжү§иЎҢе®Ңз¬¬дәҢиЎҢд»ҘеҗҺпјҢзі»з»ҹжүҚдјҡеҺ»жү§иЎҢвҖқд»»еҠЎйҳҹеҲ—вҖқдёӯзҡ„еӣһи°ғеҮҪж•°гҖӮ

жҖ»д№ӢпјҢsetTimeout(fn,0)зҡ„еҗ«д№үжҳҜпјҢжҢҮе®ҡжҹҗдёӘд»»еҠЎеңЁдё»зәҝзЁӢжңҖж—©еҸҜеҫ—зҡ„з©әй—Іж—¶й—ҙжү§иЎҢпјҢд№ҹе°ұжҳҜиҜҙпјҢе°ҪеҸҜиғҪж—©еҫ—жү§иЎҢгҖӮе®ғеңЁвҖқд»»еҠЎйҳҹеҲ—вҖқзҡ„е°ҫйғЁж·»еҠ дёҖдёӘдәӢд»¶пјҢеӣ жӯӨиҰҒзӯүеҲ°еҗҢжӯҘд»»еҠЎе’ҢвҖқд»»еҠЎйҳҹеҲ—вҖқзҺ°жңүзҡ„дәӢд»¶йғҪеӨ„зҗҶе®ҢпјҢжүҚдјҡеҫ—еҲ°жү§иЎҢгҖӮ

HTML5ж ҮеҮҶи§„е®ҡдәҶsetTimeout()зҡ„з¬¬дәҢдёӘеҸӮж•°зҡ„жңҖе°ҸеҖјпјҲжңҖзҹӯй—ҙйҡ”пјүпјҢдёҚеҫ—дҪҺдәҺ4жҜ«з§’пјҢеҰӮжһңдҪҺдәҺиҝҷдёӘеҖјпјҢе°ұдјҡиҮӘеҠЁеўһеҠ гҖӮеңЁжӯӨд№ӢеүҚпјҢиҖҒзүҲжң¬зҡ„жөҸи§ҲеҷЁйғҪе°ҶжңҖзҹӯй—ҙйҡ”и®ҫдёә10жҜ«з§’гҖӮеҸҰеӨ–пјҢеҜ№дәҺйӮЈдәӣDOMзҡ„еҸҳеҠЁпјҲе°Өе…¶жҳҜж¶үеҸҠйЎөйқўйҮҚж–°жёІжҹ“зҡ„йғЁеҲҶпјүпјҢйҖҡеёёдёҚдјҡз«ӢеҚіжү§иЎҢпјҢиҖҢжҳҜжҜҸ16жҜ«з§’жү§иЎҢдёҖж¬ЎгҖӮиҝҷж—¶дҪҝз”ЁrequestAnimationFrame()зҡ„ж•ҲжһңиҰҒеҘҪдәҺsetTimeout()гҖӮ

йңҖиҰҒжіЁж„Ҹзҡ„жҳҜпјҢsetTimeout()еҸӘжҳҜе°ҶдәӢд»¶жҸ’е…ҘдәҶвҖқд»»еҠЎйҳҹеҲ—вҖқпјҢеҝ…йЎ»зӯүеҲ°еҪ“еүҚд»Јз ҒпјҲжү§иЎҢж Ҳпјүжү§иЎҢе®ҢпјҢдё»зәҝзЁӢжүҚдјҡеҺ»жү§иЎҢе®ғжҢҮе®ҡзҡ„еӣһи°ғеҮҪж•°гҖӮиҰҒжҳҜеҪ“еүҚд»Јз ҒиҖ—ж—¶еҫҲй•ҝпјҢжңүеҸҜиғҪиҰҒзӯүеҫҲд№…пјҢжүҖд»Ҙе№¶жІЎжңүеҠһжі•дҝқиҜҒпјҢеӣһи°ғеҮҪж•°дёҖе®ҡдјҡеңЁsetTimeout()жҢҮе®ҡзҡ„ж—¶й—ҙжү§иЎҢгҖӮ

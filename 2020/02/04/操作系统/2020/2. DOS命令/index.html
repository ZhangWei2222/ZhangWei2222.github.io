windows操作系统当中常用的DOS命令

  1. 打开DOS命令窗口
  • 快捷键：win + r，打开运行窗口
  • 输入cmd回车
  1. 查看IP地址：
  • ipconfig
  • ipconfig /all 可以查看更详细的IP信息，这种查看方式可以看到网卡的物理地址。物理地址具有全球唯一性。是在生产网卡的时候，嵌入的编号。
  1. 清屏： cls
  2. 退出DOS命令窗口： exit
  3. 从DOS命令窗口当中复制文本： 任意位置点击鼠标右键–>标记 –> 选择你要复制的文本 –> 点击鼠标右键 （此时已经到剪贴板当中了） –> 找一个位置粘贴即可。
  4. 查看两台计算机之间是否可以正常通信：
  • ping 192.168.27.23 【发送和接收数据包4次】
  • ping 192.168.27.23 -t 【一直不停的发送和接收数据包】
  • ping www.baidu.com
  1. 强行终止DOS命令窗口中正在运行的程序：ctrl + c
  2. 创建目录：mkdir abc【表示在当前所在目录下新建一个目录，起名abc】
  3. 关于目录切换命令：cd

  • cd 命令的语法格式： cd 路径

  • 自动补全：cd e 【然后按tab键，当前所在的目录下所有以e开始的目录自动补全路径，当这个自动补全的路径不是自己想要的路径，可以继续使用tab键】

  • 回到上级目录：cd .. 【..是一个路径，代表当前路径的上级路径】cd ../../../

  • 直接回到根路径：cd \

  1. 查看当前目录下所有的子文件和子目录： dir
  2. 怎么切换盘符：【不需要使用cd命令】
  • c: 回车
  • d: 回车
  • e: 回车
  • f: 回车
  1. 打开注册表： regedit
